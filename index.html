Hello! Here are some questions & answers about Python's magic methods. The goal isn't to get all the questions "right". Instead, the goal is to learn something! If you find a topic you're interested in learning more about, I'd encourage you to look it up and learn more.

What does this method do?

def __init__(self, a, b):

It "initializes" an instance of a class.

That is, it creates an instance of a class, optionally using information passed in as parameters.

What do those double underscores mean?

Those double underscores (some call them dunders) are a Pythonic convention for magic methods (some call them dunder methods).

Wait, what's a magic method?

A 'magic method' lets you work with Python's built-in syntax.

You initialize a class in Python using the syntax class = NewClass(). Overriding __init__ lets you customize that process.
